# Apache Calcite (apache-calcite)
Apache Calcite is a dynamic data management framework that provides SQL parsing, query optimization, and data federation capabilities. It serves as the SQL engine for many big data systems including Apache Hive, Druid, Flink, and others.

## APIs

### Apache Calcite
Calcite provides a Java API for SQL parsing, validation, and optimization, along with adapter APIs for connecting to heterogeneous data sources and a JDBC driver for SQL query execution.
